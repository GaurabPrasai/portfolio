# Portfolio

Personal portfolio built with React and Notion API.

**Live**: [gaurabprasai.com.np]

## Setup

```bash
npm install
```

Create `.env`:
```
REACT_APP_NOTION_API_KEY=your_api_key
REACT_APP_NOTION_DATABASE_ID=your_database_id
```

## Run

```bash
npm start
```

## Build

```bash
npm run build
```
